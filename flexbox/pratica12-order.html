order: 0

Por padrão, o order é zero.
Isso significa que não muda nada, o conteúdo é apresentado conforme foi escrito.

order: diferentes

Com o order posso mudar a ordem de exibição dos elementos usando o CSS.
Neste exemplo: 4: order 1, 3: order 2, 1 e 2: order 3.
Note que elementos com o mesmo order (ex: 1 e 2) são agrupados um após o outro.

order: valores negativos

Quanto mais negativo o order, mais à frente o item é exibido.
Neste exemplo: item 3: order -2, item 4: order -1, item 2: order 2, item 1: order 1

1
2
3
4

Em resumo:
Valores positivos: quanto maior, mais adiante é exibido.
Valores negativos: quanto maior, mais à frente é exibido o item.
